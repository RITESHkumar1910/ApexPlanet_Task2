# SQL for Data Extraction and Business Analysis

## Project Overview

This project focuses on extracting and analyzing business data using SQL. The analysis was performed on the Superstore dataset to answer key business questions related to sales performance, customer behavior, and profitability.

## Objectives

- Extract meaningful information from structured data using SQL
- Analyze sales and profit trends across different dimensions
- Identify top-performing customers and regions
- Develop SQL querying skills for business analytics

## Tools and Technologies

- MySQL
- SQL
- Jupyter Notebook
- Python
- Superstore Dataset

## Analysis Performed

- Top 5 Customers by Sales
- Most Profitable Region
- Sales by Customer Segment
- Overall Sales Analysis
- Profitability Insights

## SQL Concepts Used

- SELECT
- WHERE
- GROUP BY
- ORDER BY
- HAVING
- LIMIT
- Aggregate Functions
- JOINS
- Subqueries

## Conclusion

This project demonstrates how SQL can be used to transform raw business data into meaningful insights and support data-driven business decisions.
